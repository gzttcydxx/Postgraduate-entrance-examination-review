    第一章 函数 极限 连续¶

    第一节 函数¶

    定理¶

    1. 设f(x)可导，则f(x)是奇函数\Rightarrow f'(x)是偶函数；f(x)是偶函数\Leftrightarrow f'(x)是奇函数
    2. 连续的奇函数其原函数都是偶函数；连续的偶函数其原函数之一是奇函数
    3. 若f(x)以T为周期，则f(ax+b)以\dfrac{T}{|a|}为周期
    4. 若f(x)是周期函数，则f'(x)也是周期函数；反之不一定成立
    5. 设f(x)连续且以T为周期，则F(x) = \int_0^xf(t)dt是以T为周期的周期函数\Leftrightarrow \int_0^Tf(x)dx = 0
    6. 周期函数的原函数是周期函数\Leftrightarrow其在一个周期上的积分为零
    7. f'(x)在区间I（有限）上有界\Rightarrow f(x)在I上有界

    第三章 一元函数积分学¶

    第一节 不定积分¶

    1. 函数连续\Rightarrow函数存在原函数，反之不一定成立。（若函数有第一类间断点，则其没有原函数；若函数有第二类间断点，则其可能有原函数，例如F(x)=\left\{\begin{aligned}&x^2\sin{\dfrac{1}{x}},&x\neq0\\&0,&x=0\end{aligned}\right.求导后的函数为f(x)）

    2. 若R(-\sin x, \cos x)=-R(\sin x, \cos x)，则令u=\cos x

    若R(\sin x, -\cos x)=-R(\sin x, \cos x)，则令u=\sin x

    若R(-\sin x, -\cos x)=R(\sin x, \cos x)，则令u=\tan x

    第二节 定积分¶

    1. 可积\Rightarrow有界，有界不一定可积

    反例：迪利克雷函数

    1. 若函数在闭区间内仅有有限个第一类间断点，则函数可积。

    若函数有界，且有有限个间断点，则函数可积。

    1. \int_0^\pi xf(\sin{x})dx=\dfrac{\pi}{2}\int_0^\pi f(\sin{x})dx

    2. (\int^{\psi(x)}_{\phi(x)}f(x,t)dt)'=\int^{\psi(x)}_{\phi(x)}\dfrac{\partial(x,t)}{\partial x}dt+f(x,\psi(x))\psi'(x)-f(x,\phi(x))\phi'(x)

    3. \int_{0}^{2 \pi} \cos ^{n} x d x=\int_{0}^{2 \pi} \sin ^{n} x d x=\left\{\begin{array}{ll}0, & n \text { 为奇数, } \\ 4 \int_{0}^{\frac{\pi}{2}} \sin ^{n} x d x, & n \text { 为偶数. }\end{array}\right.

    4. 做区间不动的变量代换：\int_{a}^{b} f(x) d x \overset{x=a+b-t}{===}\int_{a}^{b} f(a+b-t) d t

    5. 设 f(x) 在 [a, b] 上可积, 则 \int_{a}^{x} f(t) \mathrm{d} t 在 [a, b] 上连续.

    设 f(x) 在 [a, b] 上连续, 则 \int_{a}^{x} f(t) \mathrm{d} t 在 [a, b] 上可导且 \left(\int_{a}^{x} f(t) \mathrm{d} t\right)^{\prime}=f(x).

    设 f(x) 在 [a, b] 上存在跳跃间断点, 则 \int_{a}^{x} f(t) \mathrm{d} t 在 [a, b] 上连续但不可导.

    1. 经典错误：考虑f(x, y)=x y+(1+\alpha)\left(x^{2}+y^{2}\right)^{2}，从而, f(x, y) 在 (0,0) 点的某去心邨域内的函数值的正负由 x y 所确定,而 x y 在 (0,0) 点的去心邻域内可正可负, 则 f(x, y) 也可正可负, 又 f(0,0)=0, 则 f(x, y) 在 (0,0) 点不取得极值. 以上解法的依据是当 (x, y) \rightarrow(0,0) 时, \left(x^{2}+y^{2}\right)^{2} 是 x y 的高阶无穷小,即\lim _{x \rightarrow 0 \atop y \rightarrow 0} \frac{\left(x^{2}+y^{2}\right)^{2}}{x y}=0 但事实上极限 \lim _{x \rightarrow 0} \frac{\left(x^{2}+y^{2}\right)^{2}}{x y} 不存在! 这是由于\lim _{y=x^{4} \atop x \rightarrow 0} \frac{\left(x^{2}+y^{2}\right)^{2}}{x y}=\lim _{x \rightarrow 0} \frac{\left(x^{2}+x^{8}\right)^{2}}{x^{5}}=\lim _{x \rightarrow 0} \frac{x^{4}+2 x^{10}+x^{16}}{x^{5}}=\infty

    正确解法：

    1. 对于一元函数，在区间内有唯一极值点，则该点为最值点；而对于多元函数，在区间内有唯一极值点，该点不一定是最值点，因为其可以通过类似马鞍面的形状，找到比该极值点还要大的点（马鞍面的鞍点不是极值点）。
