## 三个一组的词

`毛泽东思想活的灵魂`
: 
    1. 实事求是
    2. 群众路线
    3. 独立自主

`工农武装割据要处理好的三个问题`
: 
    1. 土地革命（ **基本内容** ）
    2. 武装斗争（ **主要形式** ）
    3. 农村革命根据地建设（ **依托** ）

`新民主主义革命的三大法宝`
: 
    1. 统一战线
    2. 武装斗争
    3. 党的建设
    
`党的三大优良作风`
: 
    1. 理论联系实际
    2. 密切联系群众
    3. 批评与自我批评

`中国梦的科学内涵`
: 
    1. 国家富强（ **基础和保障** ）
    2. 民族振新（ **基础和保障** ）
    3. 人民幸福（ **题中之义、必然要求、出发点和落脚点** ）

`新时代党和政府的三大历史任务`
: 
    1. 祖国统一
    2. 社会主义现代化
    3. 世界和平
